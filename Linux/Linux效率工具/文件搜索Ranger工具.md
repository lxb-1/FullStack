### :page_with_curl: Ranger简介

👨‍💻 *Ranger是一个能够在终端中进行文件搜索、查找、预览、编辑等操作的文件管理器，其快捷键与Vim相类似。*

在Ubuntu上非常容易安装：

```bash
sudo apt install ranger
```
